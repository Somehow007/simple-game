# 数独游戏 - 使用手册

## 快速开始

### 环境要求

- Node.js >= 20.0.0
- pnpm >= 9.1.0

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
cd apps/web
pnpm dev
```

浏览器访问 http://localhost:3000

### 构建生产版本

```bash
# 构建所有包
pnpm build

# 仅构建 Web 版
cd apps/web
pnpm build
```

### 预览生产构建

```bash
cd apps/web
pnpm preview
```

## 游戏操作指南

### 基本操作

| 操作 | 方法 |
|------|------|
| 选中单元格 | 点击网格中的任意单元格 |
| 填入数字 | 先选中单元格，再点击下方数字面板的数字 |
| 擦除数字 | 选中单元格后，点击「⌫ 擦除」按钮 |
| 切换笔记模式 | 点击「✏️ 笔记开/关」按钮 |
| 添加候选数 | 在笔记模式下，选中空单元格后点击数字 |

### 工具栏功能

| 按钮 | 功能 |
|------|------|
| ↩️ 撤销 | 撤销上一步操作 |
| ↪️ 重做 | 恢复已撤销的操作 |
| 💡 提示 | 自动填入一个正确数字 |
| ⏸️ 暂停 | 暂停游戏并模糊网格（计时器同时暂停） |
| ▶️ 继续 | 恢复游戏 |

### 难度选择

工具栏底部有 4 个难度按钮，点击即可开始对应难度的新游戏：

| 难度 | 给定数字 | 适合人群 |
|------|---------|---------|
| 简单 | 36-45 个 | 入门用户 |
| 中等 | 30-35 个 | 休闲玩家 |
| 困难 | 27-29 个 | 进阶玩家 |
| 专家 | 22-26 个 | 资深玩家 |

### 网格高亮说明

| 高亮颜色 | 含义 |
|---------|------|
| 蓝色背景 | 当前选中的单元格 |
| 浅蓝背景 | 与选中单元格同行、同列或同宫的单元格 |
| 紫色背景 | 与选中单元格数字相同的单元格 |
| 红色背景 | 填入了错误数字的单元格 |

### 笔记模式

笔记模式用于在空单元格中标记候选数字：

1. 点击「✏️ 笔记关」按钮切换为「✏️ 笔记开」
2. 选中一个空单元格
3. 点击数字面板中的数字，该数字会作为候选数显示在单元格内
4. 再次点击同一数字可取消候选
5. 切换回普通模式后可正常填入数字

**自动清除笔记**：设置中开启后，填入数字时会自动移除同行、同列、同宫中的对应候选数。

## 设置说明

点击顶部导航栏的「⚙️ 设置」进入设置页面。

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 主题 | 亮色 / 暗色 | 亮色 |
| 高亮错误 | 是否用红色标记填错的数字 | 开启 |
| 高亮相同数字 | 是否高亮与选中格相同数字的格子 | 开启 |
| 自动清除笔记 | 填入数字时自动移除相关候选数 | 开启 |
| 显示计时器 | 是否显示游戏计时器 | 开启 |

所有设置自动保存到浏览器本地存储，刷新页面后保持不变。

---

## 统计说明

点击顶部导航栏的「📊 统计」进入统计页面。

| 统计项 | 说明 |
|--------|------|
| 已玩局数 | 总共开始的游戏数 |
| 胜利局数 | 成功完成的游戏数 |
| 胜率 | 胜利局数 / 已玩局数 |
| 当前连胜 | 连续完成的局数 |
| 最佳连胜 | 历史最高连胜数 |
| 最佳时间 | 每个难度的最快完成时间 |
| 难度分布 | 各难度已玩局数的可视化柱状图 |

所有统计数据自动保存到浏览器本地存储。

## PWA 安装

### 桌面浏览器

1. 访问游戏网页
2. 点击浏览器地址栏右侧的安装图标
3. 点击「安装」确认
4. 安装后可从桌面快捷方式启动，获得类似原生应用的体验

### 手机浏览器

**iOS (Safari)**：
1. 点击底部分享按钮
2. 选择「添加到主屏幕」
3. 点击「添加」

**Android (Chrome)**：
1. 点击浏览器菜单
2. 选择「安装应用」或「添加到主屏幕」

安装后可离线游玩。

## 常见问题

### Q: 游戏数据存在哪里？
A: 所有数据（设置、统计）存储在浏览器的 localStorage 中。清除浏览器数据会丢失游戏记录。

### Q: 支持离线游玩吗？
A: 支持。首次访问后，Service Worker 会缓存所有资源，之后可离线打开游玩。

### Q: 如何切换暗色模式？
A: 点击顶部「⚙️ 设置」，在主题选项中选择「🌙 暗色」。

### Q: 提示功能是怎么工作的？
A: 如果当前选中的单元格未填入正确答案，提示会填入该格的正确数字；否则随机选择一个未完成的格子填入正确答案。

### Q: 错误计数是怎么计算的？
A: 每次填入的数字与正确答案不同时，错误计数 +1。擦除数字不会减少错误计数。
